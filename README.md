# AppCompiler 🔧

Natural Language → Structured App Config

## What it does
Converts plain English app descriptions into 
complete JSON configs including:
- Database schema
- API endpoints
- UI structure  
- Auth & roles

## Pipeline Stages
1. Intent Extraction
2. System Design
3. Schema Generation
4. Refinement + Validation

## Setup
pip install -r requirements.txt
python server.py
